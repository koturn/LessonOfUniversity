[ 1. プログラムの概要 ]
今回作成したプログラムは、
  1. テトリス
  2. タイピングゲーム
  3. 計算ゲーム
  4. 数取りゲーム
の4つである。これらのプログラムはそれぞれ、
  tetris/
  type-game/
  calc-game/
  number-game/
というディレクトリにある。
各ゲームについての詳細は、各ディレクトリ内にあるReadMe.txtで述べる。




[ 2. コンパイル ]
このディレクトリ内にあるMakefileを使ってビルドすれば、全てのプログラムのビルド
が一度に可能である。
これは、makeの中で再帰的にmakeを呼び出すことで実現している。
実質的なMakefileは、各ネットワークゲームのディレクトリにある。

ビルドの手順は以下の通りである。
  $ make
また、オブジェクトファイルと、実行ファイルを消去する場合は、
  $ make clean
とするとよい。
ソースコードとバイナリの配布用に、オブジェクトファイルのみをクリーンアップする
ことも可能で、
  $ make objclean
とするとよい。

なお、Makefile.cygwinはCygwin用のMakefileである。
通常のMakefileと異なる点は、
  1. 実行バイナリのサフィックスとして、".exe"を採用したこと
  2. 必要なインクルードパスを追加したこと
の2点である。
Cygwin用の実質的なMakefileも各ディレクトリに配置してある。
